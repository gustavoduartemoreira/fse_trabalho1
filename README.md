## Tentando rodar o projeto 

Subir o servidor central e depois distribuido:

Subir central:
```
        git clone 
        cd fse_trabalho1
        cd central
        make clean
        make
        make run
```
Subir Distribuido:

```
        git clone 
        cd fse_trabalho1
        cd distribuido
        make clean
        make
        make run
```
